# M2AI — Methodology-to-AI Research Portal

Исследовательский портал проекта **«Методология» (М2)**: систематическая обработка наследия Г.П. Щедровицкого и построение формального алфавита операций мышления. Портал отдаёт результаты двум типам клиентов — людям (браузер, REST API) и AI-агентам (MCP через SSE).

> Статус: контент и спецификации залиты, реализация сервера — за фабрикой разработки (Jira **KAN-258**, лейбл `devin`).

## Что уже в репозитории


## Что предстоит построить (см. PROJECT-SPEC.md, разделы 7 и 11)

Сервер на **FastAPI (Python 3.11+)** + **MCP Server (SSE)**, без БД — данные читаются из `content/` в память при старте. Деплой на **Railway** через Dockerfile.

- `server/` — FastAPI app, REST API `/api/v1/*` (12 endpoints)
- `mcp_server/` — MCP server, `/mcp/` SSE endpoint (11 tools, 3 resources)
- `frontend/` — статический landing + навигация, `/viz/*` визуализации
- `Dockerfile`, `railway.json`, `requirements.txt`, CI

## Ключевые научные результаты

18 типов операций мышления (A01–A18); таксономия сопоставления S1–S6; генетическая карта (DAG 18 узлов / 21 ребро); центральный хаб A12 (PRINCIPLE_REPLACEMENT); инвариантная триада S2+S5+S6, стабильная 1956–1987; валидация на контрольном фрагменте — PASSED.
